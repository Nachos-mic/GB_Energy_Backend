# API do zadania rekrutacyjnego.

API zostało zaimplementowane w stacku: Express/Node/Typescript

Zgodnie z wymaganiami w zadaniu aplikacja posiada:

- endpoint do zczytywania danych o generationMix na trzy dni: dziś/jutro/pojutrze
- endpoint do wyliczania optymalnego okna do ładowania z prognozowanych dni: jutro/pojutrze

Aplikacja posiada także testy serwisów napisane przy użyciu biblioteki Jest
